# Code Function
Extract coding sequence from gene ID and create excel file containing codon table, RSCU and CAI values.

# Requirements
Pandas
```
  pip install pandas
```
Requests
```
  pip install requests
```
Biopython
```
  pip install biopython
```
Openpyxl 
```
  pip install openpyxl
```
Cai
```
  pip install CAI
```
